# Hydrogen Guides

This directory contains implementation guides and best practices for common Hydrogen features.

## Available Guides

### [Connect Your Store](./connectStore/connectStore.md)

**Conectar tu tienda Shopify** (Bilingual: English/Spanish)

Learn how to connect your Hydrogen project to a Shopify store using the Shopify CLI. This guide covers:

- Linking your project to a Hydrogen storefront
- Managing environment variables
- Switching between stores
- Troubleshooting common connection issues

_Aprende cómo conectar tu proyecto Hydrogen a una tienda Shopify usando Shopify CLI._

### [Search](./search/search.md)

Implement a traditional search flow using the Storefront API search endpoint. This guide includes:

- Search route implementation
- Search form component
- Search results display
- Query and fragment examples

### [Predictive Search](./predictiveSearch/predictiveSearch.md)

Add predictive (autocomplete) search functionality to your Hydrogen storefront. This guide covers:

- Predictive search API integration
- Real-time search suggestions
- Search result previews
- Performance optimization

### [Deploy to Vercel](./deployVercel/deployVercel.md)

**Desplegar en Vercel** (Spanish)

Guía completa en español para desplegar tu tienda Hydrogen en Vercel. Esta guía cubre:

- Preparación del proyecto y variables de entorno
- Configuración de Vercel
- Despliegue paso a paso
- Solución de problemas comunes
- Dominios personalizados
- Optimizaciones de producción

_Complete Spanish guide for deploying your Hydrogen store to Vercel._

## Contributing

If you have implemented a useful feature in your Hydrogen project and want to share it with the community, consider contributing a guide following the same pattern as the existing guides.

Each guide should include:

1. A clear description of what the feature does
2. Step-by-step implementation instructions
3. Code examples with explanations
4. Common troubleshooting tips
5. Links to relevant documentation
